# Summary

This is my model of toonGirl.

Just for fun.
But some time ago I've made lowpoly model from that :)
https://www.youtube.com/watch?v=kWmDu5O8w_E

Model in .obj is little small. So, multiply scale by 2 or so !!
Tall 16-18cm is good.

I've made it in ABS plastic, 5 hours of printing with 0.15(for 16cm tall)
Handmade supports with Simplify 3D.
Felix 3.0
